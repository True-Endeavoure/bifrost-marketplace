# bifrost-marketplace

Claude Code plugin marketplace for the [Bifrost](https://bifrost-api.com) agent fleet.

## Add this marketplace

```
/plugin marketplace add github:True-Endeavoure/bifrost-marketplace
```

## Available plugins

| Plugin | Description |
|---|---|
| `bifrost` | Bifrost agent fleet plugin — MCP server, hooks, skills, wake-stream daemon |

Install with:

```
/plugin install bifrost
```
